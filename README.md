# 🎮 Tic-Tac-Toe AI (Unbeatable)
**under preview/**
## 📌 Overview

This project is an AI-powered Tic-Tac-Toe game where the player competes against an unbeatable computer opponent. The AI uses the **Minimax Algorithm** to ensure it never loses.

---

## 🚀 Features

* 🎯 Play against an unbeatable AI
* 🧠 Smart decision-making using Minimax
* ⚡ Instant move response
* 💻 Responsive design (works on desktop & mobile)
* 🌐 Deployed online using Vercel
* 📶 Offline support with Service Worker

---

## 🤖 AI Logic

The AI uses the **Minimax Algorithm**, which:

* Explores all possible game states
* Maximizes its chances of winning
* Minimizes opponent's chances
* Guarantees the AI never loses
